# RfCat (command line)

```shell
# show rfcat help (optional)
$ sudo rfcat -h

# show rfcat_bootloader help (optional)
$ rfcat_bootloader --help

# run rfcat in terminal (start ipython terminal)
$ sudo rfcat -r

# select dongle
$ sudo rfcat -i [n] -r
```
